# Listar Todos os Clientes — GET /api/customers

**GET /api/customers — Recupera todos os clientes cadastrados na sua conta CertoPay. Retorna um array de objetos de cliente com detalhes completos.**

Use este endpoint para recuperar todos os clientes cadastrados na sua conta CertoPay. A resposta é um array de objetos de cliente, cada um contendo os mesmos campos retornados quando um cliente é criado pela primeira vez. É útil para construir painéis, reconciliar dados de compradores ou localizar o UUID de um cliente antes de realizar um pedido.

---

## Requisição

| Parâmetro | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `X-Api-Key` | cabeçalho | string | ✅ | Sua chave secreta de API CertoPay. Inclua em toda solicitação como `X-Api-Key: sk_live_sua_chave_aqui`. |

Este endpoint não aceita corpo na requisição nem parâmetros de consulta.

---

## Resposta

Uma solicitação bem-sucedida retorna **HTTP 200 OK** com um array de objetos de cliente. Cada objeto no array possui a seguinte estrutura:

| Campo | Tipo | Descrição |
|---|---|---|
| `id` | string | UUID único que identifica o cliente. |
| `name` | string | Nome completo do cliente. |
| `email` | string | Endereço de e-mail do cliente. |
| `document` | string | CPF do cliente (11 dígitos). |
| `phone` | string | Número de telefone do cliente. |
| `createdAt` | string | Timestamp ISO 8601 de quando o registro do cliente foi criado. |

---

## Exemplo

```bash
curl https://v2.certopaybrasil.com/api/customers \
  -H "X-Api-Key: sk_live_sua_chave_aqui"
```
