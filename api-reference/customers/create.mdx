# Criar Novo Cliente — Guia POST /api/customers

**POST /api/customers — Cadastre um novo comprador no CertoPay. Aceita nome, e-mail, documento (CPF) e telefone. Retorna o ID do cliente para uso em pedidos.**

Use este endpoint para cadastrar um novo cliente na sua conta CertoPay. Um registro de cliente armazena as informações pessoais do comprador — nome completo, endereço de e-mail, CPF e telefone — e deve ser criado antes de você realizar um pedido em nome dele. O `id` retornado é o UUID do cliente que você referenciará em todas as solicitações posteriores de pedidos e pagamentos.

---

## Requisição

| Parâmetro | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `X-Api-Key` | cabeçalho | string | ✅ | Sua chave secreta de API CertoPay. Inclua em toda solicitação como `X-Api-Key: sk_live_sua_chave_aqui`. |
| `name` | corpo | string | ✅ | Nome legal completo do cliente (ex.: `"João da Silva"`). |
| `email` | corpo | string | ✅ | Endereço de e-mail válido do cliente. Usado para recibos e notificações (ex.: `"joao@email.com"`). |
| `document` | corpo | string | ✅ | CPF — exatamente 11 dígitos numéricos sem pontuação (ex.: `"12345678900"`). |
| `phone` | corpo | string | ✅ | Número de telefone no formato brasileiro (ex.: `"(11) 99999-9999"`). |

---

## Resposta

Uma solicitação bem-sucedida retorna **HTTP 201 Created** com o objeto do cliente recém-criado.

| Campo | Tipo | Descrição |
|---|---|---|
| `id` | string | Identificador único universal (UUID) atribuído ao cliente. Guarde este valor — ele é obrigatório ao criar pedidos. |
| `name` | string | Nome completo conforme fornecido na requisição. |
| `email` | string | Endereço de e-mail conforme fornecido na requisição. |
| `document` | string | CPF conforme fornecido na requisição. |
| `phone` | string | Número de telefone conforme fornecido na requisição. |
| `createdAt` | string | Timestamp ISO 8601 registrando quando o cadastro do cliente foi criado (ex.: `"2026-06-26T10:00:00Z"`). |

---

## Exemplo

```bash
curl -X POST https://v2.certopaybrasil.com/api/customers \
  -H "X-Api-Key: sk_live_sua_chave_aqui" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "João da Silva",
    "email": "joao@email.com",
    "document": "12345678900",
    "phone": "(11) 99999-9999"
  }'
```

```json
{
  "id": "uuid-do-customer",
  "name": "João da Silva",
  "email": "joao@email.com",
  "document": "12345678900",
  "phone": "(11) 99999-9999",
  "createdAt": "2026-06-26T10:00:00Z"
}
```
