# Consultar Cliente por UUID — GET /api/customers/{id}

**GET /api/customers/{id} — Busca um único cliente pelo seu UUID. Retorna o registro completo do cliente incluindo nome, e-mail, documento e telefone.**

Use este endpoint para buscar o registro completo de um único cliente pelo seu UUID. É útil quando você precisa verificar os dados de um cliente antes de processar um pedido, exibir informações do comprador no seu painel ou confirmar que um cadastro foi salvo corretamente. Passe o UUID do cliente na URL onde `{id}` aparece.

---

## Requisição

| Parâmetro | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `X-Api-Key` | cabeçalho | string | ✅ | Sua chave secreta de API CertoPay. Inclua em toda solicitação como `X-Api-Key: sk_live_sua_chave_aqui`. |
| `id` | caminho (path) | string | ✅ | O UUID do cliente a ser consultado. É o `id` retornado quando o cliente foi criado via `POST /api/customers`. |

Este endpoint não aceita corpo na requisição.

---

## Resposta

Uma solicitação bem-sucedida retorna **HTTP 200 OK** com o objeto completo do cliente.

| Campo | Tipo | Descrição |
|---|---|---|
| `id` | string | UUID único que identifica o cliente. |
| `name` | string | Nome completo do cliente. |
| `email` | string | Endereço de e-mail do cliente. |
| `document` | string | CPF do cliente (11 dígitos). |
| `phone` | string | Número de telefone do cliente. |
| `createdAt` | string | Timestamp ISO 8601 de quando o registro do cliente foi criado. |

---

## Exemplo

```bash
curl https://v2.certopaybrasil.com/api/customers/uuid-do-customer \
  -H "X-Api-Key: sk_live_sua_chave_aqui"
```

```json
{
  "id": "uuid-do-customer",
  "name": "João da Silva",
  "email": "joao@email.com",
  "document": "12345678900",
  "phone": "(11) 99999-9999",
  "createdAt": "2026-06-26T10:00:00Z"
}
```
