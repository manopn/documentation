# Listar Todos os Pedidos — GET /api/orders

**GET /api/orders — Recupera todos os pedidos da sua conta CertoPay. Retorna um array de objetos de pedido incluindo status, valor e ID do cliente.**

Use este endpoint para recuperar todos os pedidos cadastrados na sua conta CertoPay. A resposta é um array de objetos de pedido, cada um exibindo o cliente vinculado, o valor da cobrança em centavos, o status atual do pagamento e o timestamp de criação. É útil para construir painéis de relatórios, monitorar cobranças pendentes ou conciliar pagamentos com o seu próprio sistema de gestão de pedidos.

---

## Requisição

| Parâmetro | Local | Tipo | Obrigatório | Descrição |
|---|---|---|---|---|
| `X-Api-Key` | cabeçalho | string | ✅ | Sua chave secreta de API CertoPay. Inclua em toda solicitação como `X-Api-Key: sk_live_sua_chave_aqui`. |

Este endpoint não aceita corpo na requisição nem parâmetros de consulta.

---

## Resposta

Uma solicitação bem-sucedida retorna **HTTP 200 OK** com um array de objetos de pedido. Cada objeto no array possui a seguinte estrutura:

| Campo | Tipo | Descrição |
|---|---|---|
| `id` | string | UUID único que identifica o pedido. |
| `customerId` | string | UUID do cliente vinculado a este pedido. |
| `amount` | integer | Valor da cobrança em centavos (ex.: `29700` = R$ 297,00). |
| `status` | string | Status atual do pagamento do pedido. Valores possíveis: `"PENDING"`, `"PAID"`, `"CANCELLED"`, `"EXPIRED"`. |
| `createdAt` | string | Timestamp ISO 8601 de quando o pedido foi criado. |

---

## Exemplo

```bash
curl https://v2.certopaybrasil.com/api/orders \
  -H "X-Api-Key: sk_live_sua_chave_aqui"
```
